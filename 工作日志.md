# 工作日志

## 项目概况

静态日历页面，部署于 Cloudflare Pages，绑定域名 gyf0613.xyz。
项目名：岁月履痕（Gentle Year Footprint）

## 项目文件

- `index.html` — 页面结构
- `style.css` — 样式（含主题配色）
- `script.js` — 日历渲染 + 主题切换逻辑

## 已完成功能

### v1.0（初始）
- 日历展示（月/周/年视图切换，今天高亮，回到今天按钮）
- 农历日期展示（1900-2100）
- 传统节日标注（春节、端午、中秋、元宵、七夕、重阳、除夕等）
- 记事便签（localStorage 存储，左上角黄色星号标记）
- 亮色/暗色主题切换（右上角太阳图标，自动保存）
- 页面切换动效
- Cloudflare Pages 自动部署

### v2.0（功能扩展）
- **三页面导航**：首页 / 日历 / 关于，底部导航栏切换
- **配色方案**：紫色渐变风格
- **首页概览**：
  - 大日期显示（渐变背景英雄区）
  - 本周日程卡片（按优先级显示色条）
  - 今日小建议（随机每日一条，可刷新换一条）
  - 本周概览（横向滚动，有点标记有日程的日期）
  - 浮动添加按钮（FAB）
- **日程管理**（替代原有便签）：
  - 标题 + 时间 + 优先级（低/中/高）+ 备注
  - 日历日期格子下方显示优先级彩色小圆点
  - 年视图显示日程标记
  - 点击日历日期打开日程弹窗
- **关于页面**：网站说明
- **导航栏滚动隐藏**：上拉隐藏，下拉出现
- **移动端适配**：日历容器宽度自适应

## 网络代理

访问 GitHub 需开启代理，地址：

```
http://127.0.0.1:10808
```

已在 Git 全局配置中设置：

```bash
git config --global http.proxy http://127.0.0.1:10808
git config --global https.proxy http://127.0.0.1:10808
```

如需取消代理：

```bash
git config --global --unset http.proxy
git config --global --unset https.proxy
```
