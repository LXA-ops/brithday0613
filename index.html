关于本站

一个轻量的日历日程管理工具，记录生活中的温柔点滴。

功能

  • 日历月/周/年视图，农历日期与传统节日
  • 日程管理：标题、时间、优先级，彩色标记
  • 首页概览：今日日程、本周预览
  • 亮色/暗色主题切换

技术

纯静态页面，HTML + CSS + JavaScript，数据存储在浏览器本地 (localStorage)。
